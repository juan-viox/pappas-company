# Pappas Company

Cinematic website for [Pappas Company](https://pappasco.com/) — New England's specifier-first dealer for operable partitions, movable glass walls, smoke and fire protection, custom opening solutions, and athletic equipment since 1992.

**Tagline:** Where Precision Meets Possibility.

## Stack
- Single-file HTML with GSAP 3.12.5 + ScrollTrigger
- Scroll-driven canvas hero (121 frames, Seedance 2.0)
- Google Fonts (Fraunces + Inter)
- ElevenLabs Conversational AI voice agent
- Deployed on Vercel with continuous deploy from this repo

## Structure
```
index.html            # 14 sections, all inline CSS/JS
hero-video.mp4        # 5s operable glass wall in Boston boardroom
assets/
  frames/             # 121 JPEG frames for scroll hero (frame_0001.jpg → frame_0121.jpg)
  images/             # Product, markets, project imagery
```
